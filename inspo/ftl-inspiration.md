# FTL: Faster Than Light - Room/Module Design Inspiration

## Key Features:
- Grid-style room layouts
- Clear, colored compartments
- Pixel-art style graphics
- Each room represents a system/function
- Modular, interconnected design

## Room Characteristics:
- Individual rooms with distinct purposes
- Colored borders indicating doors/connections
- System power bars and status indicators
- Crew members visible in rooms
- Variety in room sizes and configurations

## Visual Language:
- Clean, readable layouts
- Color-coded systems (weapons, shields, engines, medbay, oxygen)
- Grid-based organization
- Functional design over decoration
- Clear hierarchy and purpose

## Color Palette:
- Distinct colors per system type
- Dark backgrounds with bright UI elements
- Technical, sci-fi aesthetic

## Application to Buildings:
- Think of each building as having "rooms" or sections
- Modular construction (stack rooms vertically)
- Visual variety through different "systems" (windows, doors, decorations)
- Clear structural organization
